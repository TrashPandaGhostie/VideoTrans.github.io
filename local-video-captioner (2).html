This re-renders the whole video frame-by-frame in your browser and outputs a new .webm file with the captions drawn directly onto the picture — no upload involved. It takes roughly as long as the video's own runtime, and the video will play (with sound) while it works. Requires a browser that supports captureStream (Chrome, Edge, Firefox); Safari support is limited.

Runs entirely client-side using @xenova/transformers (Whisper, WASM). The first run downloads the model to your browser cache (40–250 MB depending on size), so it takes a minute the first time and is fast after that. Audio is extracted from your video's audio track directly in the browser — very unusual codecs may fail to decode. If this page's preview sandbox blocks the model download, save this file and open it directly in your browser instead.
